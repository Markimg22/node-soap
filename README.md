# Projeto SOAP em NodeJS

## Requisitos:

- Git
- Node 12.X

---

## Rodar o projeto:

**Clonar o repositório:**
```
git clone 
```

**Instalar dependências:**<br />
Para instalar a pasta *__node_modules__* com todas as
dependências, rode o comando dentro do diretório raiz do projeto:
```
npm install
```

**Iniciar o projeto:**<br />
Para iniciar o projeto execute o comando:
```
npm start
```
